Hi, I'm Simon Howard, aka **fraggle** or **fragglet**. I'm primarily known for stuff related to Doom
(see [the Doom Wiki](https://doomwiki.org/wiki/Simon_Howard_(Fraggle)) and
[Doomworld](https://www.doomworld.com/22years/lifetime.php) for some more context). Here on Github
you'll find:

* **[Chocolate Doom](https://github.com/chocolate-doom/chocolate-doom)** (minimalist, historically accurate Doom source port) - original author / maintainer
* **[Freedoom](https://github.com/freedoom/freedoom)** (open content Doom IWAD / game) - project founder / lead
* **[WadGadget](https://github.com/fragglet/WadGadget)** (console-based Doom WAD editor)
* **[wadptr](https://github.com/fragglet/wadptr)** (WAD compression utility)
* **[Vanilla Utilities](https://github.com/fragglet/vanilla-utilities)** (DOS Doom engine utilities)
* **[DEH 9000](https://github.com/fragglet/deh9000)** (Python-based "programmer's Dehacked")

I also develop various other retro-computing things:

* **[ipxbox](https://github.com/fragglet/ipxbox)** (DOSbox IPX-over-UDP server)
* **[Lhasa](https://github.com/fragglet/lhasa)** (free decompressor for the LHa compression format)
* **[Tumba](https://github.com/fragglet/tumba)** (retro Samba fork)
* **[SDL Sopwith](https://github.com/fragglet/sdl-sopwith)** (modern port of the game [Sopwith](https://en.wikipedia.org/wiki/Sopwith_\(video_game\)))

You'll find other things listed [on my website](https://soulsphere.org/) and [on my Github site](https://fragglet.github.io/).
